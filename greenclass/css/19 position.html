position: static - 기본/ html의 흐름대로 요소배치

position: relative - static과 동일하게 흐름대로 요소배치 / 위치이동 가능 / 기준점은 나 자신의 top/right/bottom/left

position: absolute - html의 흐름에서 제외됨 / 자유롭게 요소의 위치를 배치 / 기준점은 브라우저의 top/right/bottom/left

position: fixed - absolute와 동일 / 눈에 보이는 화면 기준으로 절대위치 / 스크롤이 바뀌어도 고정

position: sticky - 평소에는 relative와 동일 / 스크롤을 내리면 알 수 있다. / 웹사이트의 네비게이션을 만들 때 주로 사용 / 해당요소가 부여된 위치 특정값에 도달하면 고정 / overflow:hidden이 적용되면 불가능 / height 값이 %일때도 불가능

z-index: 숫자 z-index는 숫자가 높은게 위로 올라온다 / 포지션이 있을 때만 작동(static 제외)
