텍스트의 다양한 이용-inline 방식

b, strong tag

특별한 의미는 없고 b tag는 그냥 굵은 글씨를 표현해주는 태그
strong tag는 중요한 텍스트를 나타내는 태그

i, em tag

특별한 의미는 없고 그냥 i tag는 글씨를 기울어지게 이탤릭체로 나타내는 태그
em tag는 강조(emphasis)하고 싶은 부분을 표현하기 위해 사용하는 태그

u, ins tag

u tag는 특별한 의미는 없고 글자에 밑줄을 그어줍니다
ins tag는 내용을 추가(insert)하는 의미로 사용하는 태그

s, del tag

s tag는 특별한 의미는 없고 글자의 중앙에 라인을 쭉 그어주는 태그
del tag는 삭제하는 부분에 사용하는 태그

예시) 제가 좋아하는 아이스크림은 민초 엄외입니다.


sup sub tag

sup tag는 글자의 윗부분에 첨자(superscript) 형태로 작성되는 태그
sub tag는 글자의 아랫부분에 첨자(subscript) 형태로 작성되는 태그

&nbsp; 는 띄어쓰기이다
예시) 32= 9    /    SAMSUNGTM    /    H2O

small, big tag

small tag는 기본 텍스트 크기 보다 작은 글씨를 표시할 때 사용
big tag는 기본 텍스트 크기 보다 큰 글씨를 표시할 때 사용

h2가 적용된 글씨입니다 여기에 small을 적용



mark tag

텍스트를 강조하거나, 하이라이트를 표시하고 싶을 때 사용

address tag

홈페이지의 footer 부분에 많이 써있는 주소, 연락처 등을 표시할 때 사용하는 태그입니다.
-address tag는 block방식의 tag이다.
