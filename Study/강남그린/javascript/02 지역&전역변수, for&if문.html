지역변수, 전역변수

function 함수( ){ }는 함수 선언식에서 호출문이 없으면 실행이 안된다

전역변수는 리본 괄호{ } 밖에 있는 변수를 말하며 지역변수와 별개의 변수이다

리본 괄호 실행문 안에서 선언된 변수명은 그 괄호 안에서만 실행되는 지역변수라고 한다

지역변수 안에서 변수가 실행되면 전역변수와 이름이 같아도 문제가 없고 지역변수 안에 var 없이 변수가 실행이 되면 전역변수에서 덮어져서 출력된다

항상 javascript는 변수명이 같으면 무조건 맨 밑에 것만 실행됨

for문(반복문)

for(변수명; 비교식; 증감식){출력문}

비교식 : <, >, <=, >=

증감식 : ++ : 1씩 증가, -- : 1씩 감소,

for문은 한 번에 하나씩 출력이 되서 반환이 되는 방식임
