var 변수

var 변수명 = 숫자;
var 변수명 = '문자';

javascript는 변수명이 같으면 무조건 맨 밑에 것만 실행됨

연산자

+: 더하기, -: 빼기, *: 곱하기, /: 나누기

var a = 1; (숫자로 인식)
var b = 2; (숫자로 인식)
var c = "3"; (문자로 인식)

script의 document.write('내용') 은 바디에 출력을 하겠다는 것과 같은 의미

따옴표를 지워야 var 변수를 불러올 수 있다. a+b=3
숫자와 문자는 더할수 없으니 a+c=13
숫자와 문자가 곱해지면 문자에 들어있는 숫자도 곱해진다.
숫자와 문자를 빼면 문자에 들어있는 숫자도 빼진다.

alert, prompt, confirm

alert(변수명)은 변수명의 문자나 숫자의 경고창을 띄우는 것
prompt(출력 글자, 속성값)는 확인 취소창을 띄우는 것
confirm(출력 글자)는 확인 취소창을 띄우는 것
