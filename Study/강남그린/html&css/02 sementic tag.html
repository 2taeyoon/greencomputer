position: absolute; 는 단독 행위를 하여 바디 기준으로 올라감. position: relative; 는 부모 기준으로 움직인다.
부모에 relative; 를 주고 자식에 absolute; 를 주면 바디 기준이 아니라 부모 기준으로 움직인다.

float은 left right 등으로 사용하여 여러개의 단락을 일자로 정렬하는 것
margin: auto; 블록 요소를 가운데 정렬할 떄 사용
line-height: 는 글자의 수직 정렬

overflow: hidden 은 가로 세로가 있으면 재단선 처럼 잘라서 사용하는 것.
가로 세로가 없으면 자식 요소에 의해서 블럭이 씌워진다

[clear: both;]는 부모를 포함하여 자식 요소나 후손 요소가 가지고 있는 높이만큼 띄어서 유지한다.
즉, 부모가 높이값이 없어도 자식과 후손 요소에 맞춰서 띄어진다. 원위치 시키는 것

태그 안에서 개별 선택 ex) div>p = div 안에 div의 자식인 p요소만 선택.
태그 안에서 전체 선택 ex) div공백p = div 안의 모든 자식 및 후손 p요소를 선택.
